# ConsolLibrary
      
      AVAILABLE COMMAND :
      
      all - show all books
      remove - delete book by name
      add  - add a new book 
      edit - update new book name 
      
      After command use a simple sign 
      
      For example 
      add : Joanne Rowling “Harry Potter”
      
      HOW TO RUN PROJECT
      
      Сhange your configuration in consol/libr/config/application.properties
      run main
