# ConsolLibrary
    AVAILABLE COMMAND :
    all - show all books;
    remove - delete book by name;
    add  - add new book;
    edit - update book name;
    After command use the following sign ":"
    
    FOR EXAMPLE :
    add : Joanne Rowling "Harry Potter"
     
    HOW TO RUN :
    change your settings at consol/libr/config/application.properties;
    run main.
